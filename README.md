# Zoo Management System

A Python exercise for Object-Oriented Programming (OOP) focusing on inheritance and polymorphism.

## About
This project was created as a laboratory exercise for an OOP course. It demonstrates the use of a base class and derived classes to simulate a simple zoo management system.

## Concepts Used
- Inheritance
- Polymorphism
- Method Overriding
- super() function

## Classes
- **Animal** – Base class with name, age, describe(), and make_sound()
- **Lion** – Inherits from Animal, has roar()
- **Bird** – Inherits from Animal, has fly()
- **Reptile** – Inherits from Animal, has crawl()

## How to Run
```bash
python zoo_management.py
```
